# onekey
爱*查批量查询目标公司所有控股子公司、投资子公司。采用多层遍历递归查询，覆盖面全包括

## Good point
+ 递归遍历收集更加准确无遗漏
+ 一键完成前期暴露面资产收集
+ 持续维护

## How to use
1. 配置文件填入Cookie与目标公司Pid值
2. 运行：python3 main.py -h
3. 程序有两种运行模式：1.常规模式，2.递归模式。常规模式适用与查询公司旗下子公司和投资公司，深度为1层。递归模式适用与深度挖掘公司旗下所有控股子公司(50%以上)，查询深度无限。
4. 运行结束会在当前目录输出excel结果
